🥗 Welcome to NutriSmart

NutriSmart is an enterprise-level nutrition intelligence system designed to help students and young people make smarter dietary choices using data analytics, food scoring algorithms, and personalised recommendations.

This system was built to address the growing issue of poor nutrition, high fast-food consumption, and lack of awareness around balanced diets in school-aged populations.

🌍 Real-World Problem

Across Australia and globally, students are increasingly consuming high-calorie, low-nutrient foods due to convenience, cost, and lifestyle pressures.

This contributes to long-term health risks such as obesity, diabetes, fatigue, and reduced academic performance. NutriSmart addresses this by translating nutritional data into simple, actionable insights.

⚙️ System Features

Explore the core components of the NutriSmart enterprise platform:

🍎 AI Food Scoring

Each food is analysed using calories, sugar, and fat to generate a health score out of 10.

🔁 Healthy Swaps

Unhealthy foods trigger smart recommendations for healthier alternatives.

📊 Data Dashboard

Visualise nutrition trends using interactive charts and comparisons.

🍽️ Meal Planning

Generate balanced daily meal structures for better dietary habits.

🎯 Why This Project Matters

NutriSmart aligns with UN Sustainable Development Goal 3: Good Health and Wellbeing.

It demonstrates how data-driven systems can be used in real-world enterprise environments to solve social and health-related problems through technology.

💡 How It Works

1. User selects a food item
2. System analyses nutritional values
3. Algorithm generates a health score
4. Recommendation engine suggests improvements
5. Data is visualised in dashboard form
